ABOUT PASTEKING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PasteKing is a small desktop tool that displays buttons the user can click to:
  - Copy text to their clipboard instantly
  - Open websites in their browser
  - Open files or folders on their PC
  - Run system commands (e.g. timed shutdown)

Everything is driven by a plain text file called data.txt.
Your output will become that file.

FOLDERS
-------
Folders group buttons together into pages. They can be nested as deep as needed.

  ### Folder Name          ← opens a folder
  ### End_Of_This_Folder   ← closes the most recently opened folder

Every opened folder MUST be closed. Think of them like brackets:
  ### Outer Folder
      ### Inner Folder
      ### End_Of_This_Folder    ← closes Inner Folder
  ### End_Of_This_Folder        ← closes Outer Folder


ITEMS
-----
Each button is one item. Items must follow this exact order:

  ---
  type: CopyText
  "Button Label"
  """Content that gets copied or used"""

Rules:
  - --- separates items (put it before each item)
  - type: must come first
  - "Label" must come second — use double quotes
  - """Content""" must come third — use triple double-quotes
  - Triple quotes allow multiple lines of content
  - Lines starting with # are comments and are ignored
  - Do NOT put --- between a folder header and its first item


ITEM TYPES
----------

  CopyText   Copies the content to the clipboard when clicked.
             Use for: names, addresses, email addresses, phone numbers,
             long prompts, CV text, NI numbers, any reusable text.

  Link       Opens the content as a URL in the default browser.
             Use for: websites, Google Docs, YouTube, LinkedIn, etc.
             Content must start with https:// or http://

  LocalPath  Opens a file or folder on the user's Windows PC.
             Use for: Desktop folders, project folders, applications.
             Example content: C:\Users\Name\Desktop\MyFolder

  Command    Runs a Windows system command.
             Use for: shutdown timers, batch commands.
             Example content: shutdown -s -t 3600
