👑 PasteKing
PasteKing is your personal command center for quick-access text, links, and system tools. It stays right where you need it, letting you copy data or trigger actions with a single click.

Quick Start
Launch: Double-click PasteKing.exe.

Explore: Click the 📜 Credits button inside the app to see detailed instructions, setup tips, and how to organize your buttons.

Manage: You can edit your buttons directly through the app using the ✏ Edit buttons, or manage your data by editing data.txt in a text editor.

What's in the folder?
PasteKing.exe       ← The main program
data.txt            ← Your personal data (edit this to add buttons!)
README.md           ← This file

Options
Handy features in the app
📌 Pin On Top: Keeps PasteKing above all other windows—perfect for filling out long forms.

✏ In-App Editor: Click the pencil icon next to any item to edit, delete, or reorder it without touching the text file.

Folder Navigation: Use the 🏠 Home and ◀ Back buttons to dive into your nested groups.

Theme Toggle: Switch between Light and Dark modes instantly via the top bar.
